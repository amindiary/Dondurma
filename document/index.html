to use it, you should have a SMTP server, you can use your gmail, yahoomail or hotmail smtp server:

Gmail

smtp_host: smtp.gmail.com
smtp_port: 465 or 587
smtp_ssl: 1

- Hotmail

smtp_host: smtp.live.com
smtp_port: 25 or 587
smtp_start_tls: 1

- Yahoo Mail Plus (currently a paid service)

smtp_host: smtp.mail.yahoo.com
smtp_port: 25

Google Map

To set positin on google map, open js/script.js file and set below configurations:

var zoom = 15;
var latitude = 41.038966;
var longitude = 28.984451;
var customMap=true;

If you want to use custom style map seet customMap variable to true, if you want Default style map, set it to false.
